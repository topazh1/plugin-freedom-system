## Context to Preserve

**Key Decisions:**
- Plugin type: Audio Effect (Reverb)
- Reverb type: Algorithmic (CPU-efficient)
- Decay range: 0.5s to 10s (logarithmic skew 0.3)
- Size: Room dimensions (0-100%, linear)
- Drive range: 0-24dB (even harmonics, wet signal only)
- Filter: DJ-style (GainKnob reference: PluginProcessor.cpp:86-130)
  - Center (0%) = bypass
  - Negative = low-pass (200Hz at -100% → 20kHz at 0%)
  - Positive = high-pass (20Hz at 0% → 10kHz at +100%)
- Filter position: PRE (Reverb→Filter→Drive) vs POST (Reverb→Drive→Filter)
- Default preset: Subtle (30% wet, 6dB drive, 2s decay, 40% size, 0% filter, POST)
- VU meter: Shows drive saturation level (-20dB to +3dB)

**Signal Flow (PRE mode):**
```
Input → Reverb → Filter → Drive → Dry/Wet Mix → Output
```

**Signal Flow (POST mode):**
```
Input → Reverb → Drive → Filter → Dry/Wet Mix → Output
```
