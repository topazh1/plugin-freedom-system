NBS DynaDrive v1.3.0 is fully implemented and installed. 5 new features and 2 bug fixes from v1.2.0.

## v1.3.0 Changes

1. **Compressor On/Off toggle** — New `comp_enable` parameter with ON/OFF button in dynamics section header. Dims knobs when off.
2. **Drive section output volume** — New `drive_out` parameter (-24 to +12 dB) with VOL OUT knob at bottom of saturation section.
3. **Compressor section output volume** — New `comp_out` parameter (-24 to +12 dB) with VOL OUT knob at bottom of dynamics section.
4. **Numerical I/O dB readouts** — Exact dB text readouts below IN and OUT meters, updated in real-time.
5. **Gain reduction visualizer** — Real-time horizontal GR bar below compressor curve (0 to -24 dB range) with dB text readout.
6. **Bypass button overlap fix** — Added `position: relative` to `#center-title-block` so bypass button stays within center section.
7. **Meter title visibility fix** — Brightened IN/OUT labels from #444 to #666.
