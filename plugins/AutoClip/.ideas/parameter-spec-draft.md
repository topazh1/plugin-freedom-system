## Parameters

### clipThreshold
- **Type:** Float
- **Range:** 0 to 100 (percentage)
- **Default:** 0
- **DSP Purpose:** Controls the level at which hard clipping occurs

### soloClipped
- **Type:** Bool
- **Range:** Off/On (false/true)
- **Default:** Off (false)
- **DSP Purpose:** Outputs only the clipped portion of signal
